# Inko

Plateforme de lecture manga en ligne, réalisée en autonomie dans le cadre de ma formation DWWM.
Architecture JS modulaire avec séparation logique métier / affichage — sans framework.

## Stack

HTML5 · CSS3 · JavaScript ES6+ · Git

## Fonctionnalités

**Lecteur**
- Lecture plein écran, navigation chapitre par chapitre, desktop & mobile
- Reprise automatique au dernier chapitre lu par série

**Catalogue**
- Fiches séries : synopsis, auteur, genre, statut de publication, note communauté
- Recherche et filtrage par genre, popularité et statut (en cours / terminé)
- Collections personnalisées

**Compte utilisateur**
- Inscription, connexion, récupération de mot de passe, profil
